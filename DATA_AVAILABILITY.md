# Data availability notes

This repository does not redistribute TCGA whole-slide images https://www.cancer.gov/ccg/research/genome-sequencing/tcga.

- TCGA data are available through the NCI Genomic Data Commons (GDC: https://portal.gdc.cancer.gov/) under applicable access and usage terms.
- Users should obtain WSIs from GDC/TCGA directly, following relevant governance and institutional requirements.
